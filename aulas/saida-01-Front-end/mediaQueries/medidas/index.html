Tipos de medidas no CSS

As unidades de medidas no css definem tamanhos relativos e absolutos para os elementos contidos na página

px - Largura fixa: 300px. Pixels são uma unidade fixa, independente do pai
em - Baseado no tamanho da fonte do elemento pai.
(30 x 16 = 480px)
• Se o tamanho não fo definido a base será 16px
rem - Baseado no tamanho da fonte do root (html)
• Se o tamanho da fonte não for definida a base será 1rem = 16px
% - Relativo ao elemento pai (50%).
• Se o pai tiver 200px o elemento terá 100px
vw - Relativo a largura da tela view width (50vw).
• 50vw = 50% da largura da tela
vh - Relativo a altura da tela view height (50vh).
• 50vh = 50% da altura da tela
